# Aurora Trails — учебный многостраничный промо-сайт

Проект выполнен на **HTML / CSS / JavaScript** без библиотек.

## Страницы
- `index.html` — главная
- `gallery.html` — галерея
- `booking.html` — бронирование и FAQ

## Что использовано из тем курса
- DOM и работа с узлами
- изменение классов и стилей через JavaScript
- создание интерактивных блоков
- события мыши
- события клавиатуры
- события формы

## Интерактивные элементы
- popup
- accordion
- carousel
- modal gallery
- анимация элементов через JS

## Клавиши
### Главная
- `T` — смена темы
- `P` — открыть popup
- `Space` — анимация hero-блока

### Галерея
- `← / →` — листать
- `G` — включить / выключить автопрокрутку
- `Escape` — закрыть modal

### Бронирование
- `R` — очистить форму
- `1 / 2 / 3` — открыть FAQ-блоки
- `Escape` — закрыть popup

## Публикация на GitHub Pages
1. Создать новый репозиторий
2. Загрузить все файлы проекта
3. В `Settings -> Pages` выбрать:
   - `Source: Deploy from a branch`
   - `Branch: main`
   - `Folder: / (root)`
4. Сохранить настройки
5. Получить ссылку на опубликованный сайт
